1. 프로젝트 개요

신규 웹 시스템 구축을 통해 공지 등록부터 발송 History 집계까지의 End-to-End 프로세스를 자동화합니다. 특히 MS Graph API를 연동하여 아웃룩 캘린더와의 실시간 동기화를 구현합니다.

주요 구축 프로세스

인프라 구축: 공지 관리 웹 시스템 및 아웃룩 연동 엔진 개발

자원 생성: 계열사별 공지 전용 자원(Resource) 계정 확보

공지 등록: 시스템 내 GUI를 통한 직관적인 공지사항 입력

승인 프로세스: 담당 매니저 승인 후 최종 발송 처리

자동화 발송: 메일 발송 및 아웃룩 캘린더 일정 자동 등록

데이터 집계: 발송 결과 Dashboard 반영 및 통계 관리

2. 기술 상세 및 연동 체계

MS Graph API 연동

독립성 확보: 사용자의 개인 일정에 간섭하지 않는 별도의 "IT 공지 자원 일정" 방식 채택

보안: SSO(Single Sign-On) 인증 기반의 안전한 API 통신 환경 구축

최적화: Graph API를 활용한 실시간 일정 생성 및 업데이트 로직 적용
